# Usage
<!-- usage -->
```sh-session
$ npm install -g nancy
$ nancy COMMAND
running command...
$ nancy (-v|--version|version)
nancy/0.0.1 darwin-x64 node-v12.12.0
$ nancy --help [COMMAND]
USAGE
  $ nancy COMMAND
...
```
